Security Knowledge Framework

OWASP-SKF is an open source security knowledgebase including manageable projects with checklists using OWASP-ASVS/OWASP-MASVS or custom security checklists, vulnerable Labs to practise security verification using SKF-Labs and OWASP-Juice-shop and best practice code examples from SKF and the OWASP-Cheatsheets explaining the secure coding principles in multiple programming languages. Use SKF to learn and integrate security by design in your software development.
